# Rust Chip-8 Emulator 🕹️

A high-performance **Chip-8 Virtual Machine** implementation written in Rust. This project serves as a deep dive into CPU architecture, memory management, and low-level bitwise operations, leveraging Rust's safety and speed.

## ⚡ Requirements
- [Rust](https://www.rust-lang.org/) (Stable 1.70+)
- Cargo (Rust's package manager)

## 🏃 Getting Started
1. Clone the repository:
   ```bash
   git clone [https://github.com/your-username/chip8-rust.git](https://github.com/your-username/chip8-rust.git)
   cd chip8-rust
   cargo run -- path/to/rom
   ```
